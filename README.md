# 🛒 Blinkit Sales Performance Dashboard

## 📊 End-to-End Data Analytics Project

This project demonstrates an end-to-end business intelligence solution for analyzing Blinkit sales performance using SQL, Excel, Power BI, and DAX.

## 🎯 Business Objective

The objective of this project is to analyze sales performance, customer behavior, product profitability, and regional trends to support business decision-making.

## 🛠️ Tools & Technologies

- SQL
- Microsoft Excel
- Power BI
- DAX
- Power Query

## 📈 Key KPIs

- Total Sales
- Total Profit
- Profit Margin
- Average Order Value
- Customer Retention
- Top Products
- Regional Performance
